Pour git :

1) Vérifier sur quelle branche on est :
- git branch

2) Créer une autre branch
- git checkout -b "nom de la branche"
==W message qui confirme que la branche a été créé :
Switched to a new branch 'feature_images'

3) Et puis refaire :
- git add .
- git commit -m ""
- git push

4) Vérifier sur git au cas ou !!!
